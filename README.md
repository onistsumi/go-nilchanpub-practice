Цели:
- закрепить базовые конструкции Go
- научиться структурировать код
- ежедневно практиковать Git/GitHub

Структура:
- 01-variables
- 02-branching
- 03-loops
- 04-functions
- 05-pointers
- 06-structs
- 07-packages
- 08-slices-maps
- 09-inputs
